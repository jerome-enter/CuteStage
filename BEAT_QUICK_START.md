# Beat 시스템 빠른 시작 가이드 🚀

## 1분 안에 이해하기

**Beat 시스템**은 연극 시나리오를 **의미 있는 단위(Beat)**로 만드는 시스템입니다.

### Before (복잡함)

```
1. 캐릭터A 왼쪽에서 등장
2. 캐릭터A 중앙으로 이동
3. 캐릭터A 멈춤
4. 캐릭터A 손 흔들기
5. 캐릭터A 대사: "안녕?"
6. 캐릭터B 오른쪽에서 등장
7. 캐릭터B 중앙으로 이동
...
```

→ 너무 복잡하고, "왜 이렇게 움직이지?" 맥락이 없음

### After (간단함)

```
Beat 1: "첫 만남"
  → 두 사람이 양쪽에서 등장하여 중앙에서 만남
  → 시스템이 자동으로 동작/대사 처리

Beat 2: "어색한 침묵"
  → 서로 쳐다보며 어색한 분위기
  
Beat 3: "고백"
  → A가 B에게 용기내어 고백
```

→ 의미 중심으로 작업, 직관적!

## 3분 튜토리얼

### Step 1: 앱에서 Beat Creator 열기

```
앱 실행 → 시나리오 선택 → [새로 만들기] 버튼
```

### Step 2: 캐릭터 추가

```
👥 등장인물 섹션
[+] 버튼 클릭
  이름: 철수
  성별: 남성
  
[+] 버튼 클릭
  이름: 영희
  성별: 여성
```

### Step 3: Beat 템플릿 선택

```
📖 Beat 템플릿 팔레트
[만남] 탭 선택
  
👋 첫 만남
  두 사람이 처음 만나는 순간
  [추가] 버튼 클릭
  
😅 어색한 침묵
  서로를 의식하며...
  [추가] 버튼 클릭
```

### Step 4: 더 많은 Beat 추가

```
[감정] 탭 선택
  
❤️ 고백
  용기를 내어 고백하는 순간
  [추가]
```

### Step 5: 저장 및 재생

```
[💾] 저장 버튼
  제목: 공원에서의 첫 만남
  설명: 철수와 영희의 운명적인 만남
  [저장]
  
시나리오 목록으로 돌아감
→ "공원에서의 첫 만남" 선택
→ [▶️] 재생!
```

## 제공되는 템플릿

### 👋 만남

- **첫 만남**: 양쪽에서 등장 → 중앙에서 만남 → 인사
- **어색한 침묵**: 서로 의식하며 긴장

### ⚡ 갈등

- **대립**: 한 명은 다가가고, 한 명은 물러남 → 긴장감
- **물러서기**: 당황하여 뒤로 물러남

### ❤️ 감정

- **고백**: 용기 내어 마음 고백 → 긴장된 순간
- **환호**: 기쁨을 표현하며 박수

### 👋 작별

- **작별 인사**: 양쪽으로 퇴장하며 손 흔들기

### 🎭 단독

- **등장**: 무대에 등장
- **독백**: 혼자 생각에 잠김

## 예시 시나리오

### 예시 1: 짧은 만남 (3개 Beat, 11초)

```
1. 첫 만남 (4초)
   철수와 영희가 양쪽에서 등장
   
2. 어색한 침묵 (3초)
   서로 쳐다보며 어색함
   
3. 작별 인사 (4초)
   "다음에 또 봐!" 하며 퇴장
```

### 예시 2: 로맨틱 스토리 (5개 Beat, 21초)

```
1. 등장 (철수) (3초)
2. 등장 (영희) (3초)
3. 첫 만남 (4초)
4. 고백 (철수 → 영희) (5초)
5. 환호 (6초)
   "와! 나도 좋아했어!"
```

### 예시 3: 갈등과 화해 (6개 Beat, 28초)

```
1. 첫 만남 (4초)
2. 대립 (5초)
   "왜 그랬어?"
3. 물러서기 (철수) (3초)
4. 어색한 침묵 (3초)
5. 고백 (철수) (6초)
   "미안해... 내가 잘못했어"
6. 환호 (7초)
   화해하며 기뻐함
```

## 자주 묻는 질문 (FAQ)

### Q: Beat를 커스터마이징할 수 있나요?

**A (현재):** Phase 1에서는 템플릿을 조합하는 방식입니다.
**A (추후):** Phase 2에서 대사 수정, 동작 조정 등이 추가될 예정입니다.

### Q: 캐릭터를 3명 이상 추가할 수 있나요?

**A:** 네! 캐릭터는 제한 없이 추가할 수 있습니다. 단, 일부 템플릿은 2명 기준으로 설계되어 있습니다.

### Q: 배경을 바꿀 수 있나요?

**A (현재):** Phase 1에서는 기본 무대 배경을 사용합니다.
**A (추후):** Phase 3에서 배경 선택/업로드 기능이 추가될 예정입니다.

### Q: 대사를 수정할 수 있나요?

**A (현재):** 템플릿의 기본 대사가 사용됩니다.
**A (추후):** Phase 2에서 Beat 에디터로 대사를 수정할 수 있습니다.

### Q: 음악이나 효과음을 넣을 수 있나요?

**A (현재):** Phase 1에서는 지원하지 않습니다.
**A (추후):** Phase 4에서 BGM/효과음 기능이 추가될 예정입니다.

### Q: Beat 순서를 바꿀 수 있나요?

**A:** 타임라인에서 Beat 카드를 드래그하여 순서를 변경할 수 있습니다. (Phase 1에서는 제거 후 다시 추가)

### Q: Beat를 저장하지 않고 바로 미리보기할 수 있나요?

**A:** [▶️ 미리보기] 버튼으로 저장 전에 테스트할 수 있습니다. (구현 예정)

### 커스텀 Beat 만들기

```kotlin
val customBeat = Beat(
    id = "custom_1",
    name = "놀라운 발견",
    description = "캐릭터가 중요한 것을 발견",
    duration = 4f,
    layers = BeatLayers(
        characters = listOf(
            CharacterAction(
                characterId = "char1",
                characterName = "철수",
                gender = CharacterGender.MALE,
                movement = Movement(
                    type = MovementType.STAY,
                    from = Position.CENTER
                ),
                emotion = Emotion(EmotionType.SURPRISED, 0.9f),
                facingDirection = Direction.FORWARD
            )
        ),
        dialogues = listOf(
            DialogueAction(
                characterId = "char1",
                text = "이게 바로 그거야!",
                emotion = EmotionType.EXCITED,
                delay = 1.5f
            )
        )
    )
)

// TheaterScript로 변환
val script = listOf(customBeat).toTheaterScript()
```

### 새 템플릿 추가하기

```kotlin
// BeatTemplates.kt에 추가
fun surprise(character: CharacterInfo, line: String): Beat {
    return Beat(
        id = "surprise_${System.currentTimeMillis()}",
        name = "깜짝 놀람",
        description = "예상치 못한 일에 놀라는 순간",
        duration = 3f,
        layers = BeatLayers(
            characters = listOf(
                CharacterAction(
                    characterId = character.id,
                    characterName = character.name,
                    gender = character.gender,
                    movement = Movement(type = MovementType.STAY, from = Position.CENTER),
                    emotion = Emotion(EmotionType.SURPRISED, 1.0f),
                    gesture = Gesture(GestureType.POINT),
                    facingDirection = Direction.FORWARD
                )
            ),
            dialogues = listOf(
                DialogueAction(
                    characterId = character.id,
                    text = line,
                    emotion = EmotionType.SURPRISED,
                    delay = 1f
                )
            )
        )
    )
}
```
