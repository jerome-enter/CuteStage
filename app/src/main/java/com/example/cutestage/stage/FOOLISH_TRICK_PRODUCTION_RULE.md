# "폭삭 속았수다 - 제1막 조기 밥상" 제작 가이드

## 📋 프로덕션 개요

- **작품명**: 폭삭 속았수다 - 제1막 "조기 밥상"
- **장르**: 제주 가족 드라마
- **배경**: 제주도 초가집 마당
- **주제**: 모성애, 가족 갈등, 계급 차별

## 🎨 리소스 명명 규칙

### 새로운 규칙

```
인물_자세_감정_소지품.png
```

**구성 요소**:

- **인물**: mother, father, daughter, grandma 등
- **자세**: sit, stand, walk, throw 등
- **감정**: angry, sad, suprise, talk 등 (선택사항)
- **소지품**: fish, basket 등 (선택사항)

**예시**:

- `mother_stand_angry_fish.png` - 광례가 화나서 서있고 조기를 들고 있음
- `mother_throw_angry_fish.png` - 광례가 화나서 조기를 던지는 모습
- `daughter_sit.png` - 애순이 앉아 있음 (감정/소지품 없음)
- `father_stand_angry.png` - 작은아버지가 화나서 서있음

## 🎭 등장인물 및 리소스 매핑

### 1. 광례 (40대, 여) - 애순의 엄마

**캐릭터 특성**:
- 물질을 하는 억척스러운 제주 해녀
- 목소리가 크고 강인함
- 딸에 대한 사랑이 지극함

**사용 리소스**:
- `mother_walk.png` - 등장 시, 이동 시
- `mother_stand_angry_fish.png` - 조기를 들고 분노 표현 시
- `mother_throw_angry_fish.png` - 조기를 내던지는 씬
- `mother_stand_talk.png` - 일반 대사 시

**음성 설정**:
```kotlin
CharacterVoice(
    pitch = 1.2f,  // 중간 높이의 여성 목소리
    speed = 80,     // 빠르고 다급하게
    duration = 50,
    volume = 0.7f   // 큰 목소리
)
```

### 2. 작은아버지 (40대, 남) - 애순의 삼촌

**캐릭터 특성**:
- 뻔뻔하고 이기적
- 가부장적 권위를 내세움
- 형수를 무시하는 태도

**사용 리소스**:
- `father_sit.png` - 평상에 앉아있는 기본 자세
- `father_sit_suprise.png` - 광례가 등장했을 때
- `father_stand_angry.png` - 화를 낼 때

**음성 설정**:
```kotlin
CharacterVoice(
    pitch = 0.7f,   // 낮고 무거운 남성 목소리
    speed = 95,     // 느리고 무겁게
    duration = 55,
    volume = 0.6f
)
```

### 3. 할머니 (70대, 여) - 애순의 친할머니

**캐릭터 특성**:
- 며느리에게 미안함
- 아들 눈치를 봄
- 무기력하고 슬픔

**사용 리소스**:
- `grandma_sit.png` - 앉아 있는 기본 자세
- `grandma_sit_sad.png` - 흐느끼며 통곡하는 씬

**음성 설정**:
```kotlin
CharacterVoice(
    pitch = 1.1f,   // 높지만 떨리는 노인 목소리
    speed = 110,    // 느리고 힘없이
    duration = 52,
    volume = 0.4f   // 작은 목소리
)
```

### 4. 애순 (10대, 여) - 딸

**캐릭터 특성**:
- 눈치 빠르고 조숙함
- 집안일을 도맡아 함
- 겁에 질려있지만 엄마를 따름

**사용 리소스**:
- `daughter_sit.png` - 수돗가에서 설거지하는 자세
- `daughter_sit_suprise.png` - 놀라거나 겁먹은 표정
- `daughter_walk.png` - 엄마와 함께 퇴장할 때

**음성 설정**:
```kotlin
CharacterVoice(
    pitch = 1.6f,   // 높고 어린 목소리
    speed = 75,     // 빠르고 불안하게
    duration = 45,
    volume = 0.45f  // 작고 떨리는 목소리
)
```

## 🎨 무대 구성

### 배경
- **리소스**: `house.png` (제주 초가집)
- **분위기**: 해 질 녘의 노르스름한 주황빛

### 무대 배치
```
┌─────────────────────────────────────┐
│                                     │
│   [할머니]                          │
│   [작은아버지]  (평상 - 상수/오른쪽)│
│                                     │
│              [광례 등장 지점]       │
│                                     │
│                    [애순]           │
│                 (수돗가 - 하수/왼쪽)│
│                                     │
└─────────────────────────────────────┘
```

### 위치 가이드 (dp 기준)
- **평상 영역 (상수/오른쪽)**: x = 200~280dp
- **수돗가 (하수/왼쪽)**: x = 30~80dp
- **무대 중앙 (등장 지점)**: x = 150dp
- **y 위치**:
    - 서있는 캐릭터: 120~150dp
    - 앉아있는 캐릭터: 160~180dp

## 📝 씬 구성 계획

### 씬 1: 평화로운(?) 저녁 풍경
- 작은아버지 평상에서 이쑤시개
- 할머니 먼 곳 응시
- 애순 수돗가에서 설거지
- **시간**: 3000ms
- **효과**: 귀뚜라미 소리 (대사로 표현)

### 씬 2: 작은아버지의 횡포
- 작은아버지 → 애순에게 호통
- 애순 반응
- **시간**: 2500ms

### 씬 3: 광례 등장 (클라이맥스 시작)
- 광례 무대 뒤에서 거칠게 등장
- 긴장감 고조
- **시간**: 2000ms
- **효과**: 무거운 발소리 (행동으로 표현)

### 씬 4: 조기를 내던지는 장면 (절정)

- 광례 평상에 조기 던짐
- 작은아버지 놀람
- **시간**: 3500ms
- **중요**: 이 씬이 작품의 핵심!
- **리소스**: `mother_throw_angry_fish.png` 사용

### 씬 5-8: 대치와 갈등
- 광례 vs 작은아버지 언쟁
- 할머니의 중재와 울음
- **각 씬**: 2500~3500ms
- **리소스**: `mother_stand_angry_fish.png` 사용 (조기를 들고 있음)

### 씬 9-10: 광례와 애순
- 광례가 애순에게 다가감
- 손을 녹이는 장면
- **시간**: 3000ms
- **감정**: 모성애의 절정

### 씬 11: 퇴장 (엔딩)
- 광례와 애순 함께 퇴장
- 할머니 통곡
- 작은아버지 등 돌림
- **시간**: 4000ms
- **마무리**: 조기에 조명 (대사로 표현)

## 🎬 연출 포인트

### 1. 거리감 표현
- 평상(어른 공간)과 수돗가(애순 노동 공간)의 **시각적 거리**를 최대한 확보
- 이를 통해 **심리적 소외감** 표현

### 2. 광례의 압도적 에너지
- 등장 시 scale 1.1~1.2 사용
- 빠른 애니메이션 (500ms 이하)
- 목소리 큼 (volume 0.7f)

### 3. 조기 소품 연출

- **`mother_throw_angry_fish.png`** 사용 - 조기를 던지는 동작
- **`mother_stand_angry_fish.png`** 사용 - 조기를 들고 화남
- 대사로 "퍽!" 소리 표현
- 작은아버지 scale 감소로 압도당함 표현

### 4. 감정선 흐름
```
평온 → 불안 → 긴장 → 폭발 → 갈등 → 모성애 → 결별
```

### 5. 제주 방언 살리기
- "~수다", "~민", "~게" 등 제주어 유지
- 타이핑 속도를 적절히 조절 (40~60ms)
